# Path-Planning & TSP Algorithms

This repo includes basic algorithm implementations for path planning and traveling salesman problem (TSP) for self-learning use. 
Some code formatting and annotations were polished by AI.

## Files
- `A401_Astar_GUIDisplay.py`: Hand-written A* algorithm, can be displayed with GUI. You can set start, end, obstacles in the GUI.  Also, you can see the calculation process and set the animation's updating speed.
